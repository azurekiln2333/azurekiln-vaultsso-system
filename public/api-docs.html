OAuth2 API Documentation

Complete reference for the VaultSSO OAuth2 provider. Implement secure authentication using industry-standard OAuth2 and OpenID Connect protocols.

GET

OpenID Discovery

Retrieve the OpenID Connect discovery document containing all endpoint URLs and supported features.

/.well-known/openid-configuration

Response

{
  "issuer": "http://localhost:3000",
  "authorization_endpoint": "http://localhost:3000/oauth2/authorize",
  "token_endpoint": "http://localhost:3000/oauth2/token",
  "userinfo_endpoint": "http://localhost:3000/oauth2/userinfo",
  "jwks_uri": "http://localhost:3000/.well-known/jwks.json",
  "response_types_supported": ["code", "token", "id_token"],
  "subject_types_supported": ["public"],
  "id_token_signing_alg_values_supported": ["RS256", "HS256"],
  "scopes_supported": ["openid", "profile", "email", "offline_access"],
  "grant_types_supported": ["authorization_code", "refresh_token", "client_credentials"]
}
GET

Authorization Endpoint

Initiate the OAuth2 authorization flow. Redirects users to the login page for authentication.

/oauth2/authorize

Parameters

Parameter Required Description
response_type Yes Must be code
client_id Yes Your client application ID
redirect_uri Yes Must match registered redirect URI
scope Yes Space-separated list of scopes
state Recommended CSRF protection token

Example Request

GET /oauth2/authorize?
    response_type=code&
    client_id=salesforce-prod&
    redirect_uri=https://login.salesforce.com/oauth2/callback&
    scope=openid%20profile%20email&
    state=xyzABC123
POST

Token Endpoint

Exchange an authorization code for access tokens. Supports multiple grant types.

/oauth2/token

Authorization Code Grant

POST /oauth2/token
Content-Type: application/x-www-form-urlencoded

grant_type=authorization_code&
code=AUTHORIZATION_CODE&
redirect_uri=https://login.salesforce.com/oauth2/callback&
client_id=salesforce-prod&
client_secret=CLIENT_SECRET

Response

{
  "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "token_type": "Bearer",
  "expires_in": 3600,
  "refresh_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "id_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "scope": "openid profile email"
}
GET

UserInfo Endpoint

Retrieve user profile information using an access token.

/oauth2/userinfo

Request

GET /oauth2/userinfo
Authorization: Bearer ACCESS_TOKEN

Response

{
  "sub": "user-001",
  "name": "Alexander Chen",
  "email": "demo@vaultsso.com",
  "picture": "https://example.com/avatar.jpg",
  "email_verified": true
}
POST

Token Introspection

Validate and retrieve information about an access token.

/oauth2/introspect

Request

POST /oauth2/introspect
Content-Type: application/x-www-form-urlencoded

token=ACCESS_TOKEN

Response

{
  "active": true,
  "sub": "user-001",
  "aud": "salesforce-prod",
  "scope": "openid profile email",
  "exp": 1705315800,
  "iat": 1705312200
}
POST

Token Revocation

Revoke an access token or refresh token.

/oauth2/revoke

Request

POST /oauth2/revoke
Content-Type: application/x-www-form-urlencoded

token=ACCESS_TOKEN

Response

Returns HTTP 200 on success with empty body.

Available Scopes

badge

openid

Required scope for OpenID Connect. Returns an ID token with user identity claims.

person

profile

Access to user's profile information including name, picture, and other profile data.

mail

email

Access to user's email address and email verification status.

sync

offline_access

Request a refresh token for long-lived access without user interaction.
